## 📡 Deployment

This site is deployed using **GitHub Pages**.

To deploy your own:
1. Push code to `main` branch.
2. Go to **Settings > Pages**.
3. Choose source: `main` branch → `/ (root)`.
4. Done! Your site will be live.
